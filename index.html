How to Install RekanIkan Fish Feeder

Jika ini adalah pertama kalinya Anda menggunakan fish feeder, proses instalasi mungkin terasa sedikit membingungkan. Namun jangan khawatir, panduan ini akan membantu Anda langkah demi langkah untuk menghubungkan alat fish feeder dengan aplikasi RekanIkan secara mudah dan cepat.

Sebelum memulai: Pastikan Anda telah mengunduh aplikasi RekanIkan melalui Google Play Store.

Langkah 1: Sambungkan Alat ke Sumber Daya Listrik

  1. Hubungkan kabel daya fish feeder ke stopkontak atau sumber listrik terdekat.
  2. Pastikan area instalasi memiliki koneksi Wi-Fi yang stabil.
  3. Saat alat dinyalakan, layar LCD akan otomatis menyala dan menampilkan status koneksi internet.

Langkah 2: Hubungkan Fish Feeder ke Wi-Fi

  1. Buka pengaturan Wi-Fi di perangkat Anda (smartphone, laptop, atau lainnya).
  2. Temukan dan hubungkan ke jaringan Wi-Fi bernama ESP8266-MQTT.
  3. Setelah terhubung, Anda akan diarahkan otomatis ke halaman pengaturan Wi-Fi (Wi-Fi Manager).
  4. Masukkan:
    • Nama jaringan (SSID) Wi-Fi rumah Anda
    • Password Wi-Fi dengan benar
  5. Tekan tombol "Simpan" atau "Connect" pada halaman tersebut.
  6. Jika koneksi berhasil, alat akan mengeluarkan bunyi “tet tet tet” sebagai tanda sudah tersambung ke internet.

Langkah 3: Daftarkan Alat di Aplikasi RekanIkan

  1. Buka aplikasi RekanIkan dan lakukan registrasi atau login ke akun Anda.
  2. Masuk ke menu Your Feeder.
  3. Tekan tombol tambah “+” (floating button biru) untuk menambahkan alat baru.
  4. Pilih salah satu metode:
    • Scan QR Code (direkomendasikan)
    • Masukkan ID Manual
  5. Jika menggunakan ID manual, masukkan ID yang tertera pada alat secara manual.
  6. Lengkapi data alat:
    • Nama alat
    • Deskripsi alat
    • Zona waktu (timezone)
    • Akuarium (opsional)
  7. Simpan data. Alat akan langsung muncul pada daftar feeder Anda di aplikasi.

Langkah 4: Uji Coba Pemberian Pakan

  1. Pastikan wadah pakan telah terisi.
  2. Dari aplikasi, lakukan eksekusi pemberian pakan untuk menguji apakah alat bekerja dengan baik.
  3. Periksa apakah pakan keluar sesuai instruksi dan perangkat merespons dengan normal.

Penutup

Selamat! Anda telah berhasil menginstal dan menghubungkan fish feeder dengan aplikasi RekanIkan. Kini Anda dapat dengan mudah mengatur jadwal pemberian pakan otomatis dan memantau aktivitasnya langsung dari smartphone Anda.
